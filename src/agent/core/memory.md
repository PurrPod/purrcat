重要技术发现
agent_vm目录是老板电脑文件系统的挂载点
execute_command工具访问沙盒环境
filesystem工具访问老板电脑环境
动态工具需要通过call_dynamic_tool进行代理调用
write_text_file等工具有沙盒白名单限制

必须严格区分沙盒环境和老板环境
沙盒环境和老板电脑文件系统需要区分操作
动态工具不能直接调用，必须通过call_dynamic_tool
文件操作有权限限制，需要在白名单路径内操作
