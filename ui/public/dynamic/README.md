# Dynamic Layout 网页源码

## 文件结构
```
dynamic_layout_website/
├── index.html                    # 主HTML文件 (4.4KB)
├── dynamic-layout-n4es4kkb.js    # 主JavaScript文件 (142KB)
├── openai-symbol-e4cvf64r.svg    # OpenAI logo (2.4KB)
├── claude-symbol-7ve4mrvd.svg    # Claude logo (4.1KB)
└── README.md                     # 说明文件
```

## 网页特点
1. **动态布局**：标题为"Dynamic Layout"
2. **JavaScript驱动**：主要内容通过JS动态生成
3. **响应式设计**：包含移动端适配
4. **交互功能**：可以水平和垂直调整大小，点击logo
5. **现代CSS**：使用CSS变量、radial-gradient等

## 技术栈
- HTML5 + CSS3 (CSS变量、媒体查询)
- JavaScript (ES模块)
- SVG图片资源

## 资源引用
- HTML中引用了 `../dynamic-layout-n4es4kkb.js`
- JS中动态加载了两个SVG logo
- 配色方案通过CSS变量定义

## 使用说明
1. 打开 `index.html` 在浏览器中查看
2. 可以水平和垂直调整浏览器窗口大小
3. 点击页面中的logo进行交互
